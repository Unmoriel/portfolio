Shape2PCS

Un convertisseur de fichiers géospatiaux réalisé durant mon stage chez Poisson Soluble

Voir des images Rapport de stage

Convertie & édite

Shape2PCS est un convertisseur de fichier shape en format geoJSON. Il permet aussi d'éditer les données contenus dans le shapefile.

Symfony

Construit avec le framework PHP Symfony ainsi que javascript et bootstrap pour le front-end

Professionel

Ce projet a été developpé durant un stage de deux mois au sein de l'entreprise Poisson Soluble à Montpellier.

Contexte du projet

Au cours de l'année, Poisson Soluble a été chargé d'intégrer des fichiers shapefile dans son application pour le compte de Predict, une entreprise qui aide les communes françaises à gérer leurs Plans Communaux de Sauvegarde (PCS). Les PCS sont des dispositifs obligatoires visant à planifier les mesures de protection et de gestion de crise en cas d'événements majeurs comme des inondations ou des incendies. Certaines communes disposent de leurs PCS sous forme de fichiers shapefile qu'elles transmettent à Predict, qui les envoie ensuite à Poisson Soluble pour intégration. Pour faciliter ce processus, Poisson Soluble a eu besoin de développer un convertisseur de fichiers shapefile en geoJSON.

Déroulement du projet :

Le projet de stage vise à concevoir une application web permettant de convertir des fichiers shapefile en fichiers geoJSON, deux formats couramment utilisés dans les systèmes d'information géographique (SIG). Les utilisateurs peuvent télécharger un fichier compressé contenant le shapefile, sélectionner les attributs à inclure dans le fichier geoJSON final, et lancer la conversion automatiquement. L'application offre la possibilité de renommer des attributs du shapefile selon des champs standards définis par Poisson Soluble, pour une utilisation optimale dans les applications web de Predict. Une fonctionnalité de prévisualisation permet de visualiser les points géographiques sur une carte avant la conversion, garantissant l'exactitude des données.
